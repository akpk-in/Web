## 1. Cloud Certifications (Start Here)

- **AWS Path**
    - AWS Solutions Architect Associate
    - AWS Solutions Architect Professional
    - AWS DevOps Engineer Professional ($300, valid 3 years)
- **Azure Path**
    - AZ-104 Administrator
    - AZ-400 DevOps Engineer Expert
- **Google Cloud Path**
    - GCP Associate Cloud Engineer
    - GCP Professional DevOps Engineer

## 2. Container Certifications

- **Docker**
    - Docker Certified Associate (DCA)
    - Focus: Container basics, Docker Enterprise, security
- **Kubernetes**
    - Certified Kubernetes Administrator (CKA)
    - Certified Kubernetes Application Developer (CKAD)

## 3. Infrastructure as Code

- **HashiCorp**
    - Terraform Associate
    - Vault Associate

## 4. CI/CD & DevOps Tools

- **Jenkins**
    - Certified Jenkins Engineer (CJE)
    - Certified CloudBees Jenkins Engineer (CCJE)
- **GitLab**
    - GitLab Certified Associate
    - GitLab Certified Professional

## 5. System Administration

- **Red Hat**
    - RHCSA (System Administrator)
    - RHCE (Certified Engineer)
    - RHCA (Certified Architect)

## Recommended Learning Path

1. Choose a cloud platform (AWS/Azure/GCP) and get certified
2. Learn containers (Docker → Kubernetes)
3. Master Infrastructure as Code (Terraform)
4. Specialize in CI/CD tools (Jenkins/GitLab)
5. Add specialized certifications based on job requirements
