Here are some practical things you can do to learn linux.

1. Install and Configure web servers (Apache, Nginx, Tomcat, etc..) and learn how web servers work.
2. Host static/Dynamic websites on web servers and play around with different configurations.
3. Setup Load balancers & Reverse Proxy (Nginx, HA proxy, etc). Understand each configuration and algorithm behind load balancing.
4. Setup and Database and understand its configurations and management. (MySQL, PostgreSQL, MongoDB, etc)
5. Break something and learn to troubleshoot. You can start with the [Sadserver scenarios](https://sadservers.com/scenarios)
6. Develop shell scripts to automate repeated tasks.
7. Try to learn the commands while working on real-world scenarios.
