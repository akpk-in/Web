Foundation Level:

- Google Workspace Administrator (though retiring soon, this represents basic cloud administration)
- Cloud Developer (fundamental cloud development skills)

Intermediate Level:

- Cloud Database Engineer (builds on development fundamentals with database specialization)
- Cloud Network Engineer (requires understanding of core infrastructure)
- Cloud DevOps Engineer (combines development and operations knowledge)

Advanced Level:

- Cloud Architect (requires broad knowledge across multiple domains)
- Cloud Security Engineer (needs deep understanding of all cloud components)
- Data Engineer (specialized knowledge in data systems and architecture)
- Machine Learning Engineer (combines advanced data skills with AI/ML expertise)
